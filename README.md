# GS Advisory — Backend API

Node.js + Express + SQLite backend for gsadvisory.in

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /api/auth/login | Login (employee/client/admin) |
| POST | /api/auth/register | Add new user (admin only) |
| GET | /api/auth/me | Get current user |
| GET | /api/clients | List all clients |
| POST | /api/clients | Add client |
| PUT | /api/clients/:id | Update client |
| GET | /api/tasks | List tasks |
| POST | /api/tasks | Create task |
| PUT | /api/tasks/:id | Update task status |
| GET | /api/invoices | List invoices |
| POST | /api/invoices | Create invoice |
| GET | /api/invoices/:id/pdf | Download invoice PDF |
| POST | /api/invoices/:id/send | Email invoice to client |
| POST | /api/documents/upload | Upload document |
| GET | /api/documents/:id/download | Download document |
| POST | /api/contact | Submit contact form |
| GET | /api/stats | Dashboard statistics |
| GET | /api/health | Health check |

## Deploy to Render

1. Push this folder to a GitHub repository
2. Go to render.com → New Web Service
3. Connect your GitHub repo
4. Set environment variables (see .env.example)
5. Deploy

## Environment Variables (set in Render dashboard)

- `JWT_SECRET` — auto-generated by Render
- `MAIL_USER` — admin@gsadvisory.in
- `MAIL_PASS` — Gmail App Password
- `ADMIN_PASSWORD` — your secure admin password
